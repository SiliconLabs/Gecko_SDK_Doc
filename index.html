Gecko SDK Documentation

Welcome to the documentation for the Gecko SDK for Silicon Labs 32-bit MCUs and SoCs. The Gecko SDK contains device header files, the EMLIB Peripheral Library, EnergyAware Driver library, middleware and support for Starter Kits and Development Kits.
